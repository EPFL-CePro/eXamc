AMC application#

First, you have to check if in the Project tab everything’s ok. Check if the to infos bar are green.

Please do not update the document after starting the AMC correction.

Import the scans#

  1. Go to the “Data capture” tab

  2. Click on the “automatic” button

  3. Upload or choose to import from app your scans

Check the boxes with the “sensitivity” setting#

This step allows you to check the boxes (a trace in a box, a cross incorrectly erased, etc.).

  1. Click on the “manual” button.

  2. In the window that opens, select “scan”.

  3. Click on the “sensibility” column to sort the copies to check.

  4. Check all those with a value greater than 0. The software counts as checked all those with a red square. To deselect a box, click on it.

  5. Click on the “marking” tab.

  6. Check the “update marking scale” box.

  7. Click on the “Mark” button.

Check the boxes with the “invalid” parameter#

Invalids are where students have selected multiple boxes for questions with only one correct answer (SCQ, True-false).

Go back to the “Data capture” tab

  1. Click on the “manual” button.

  2. Select the parameters: inv

  3. Select / deselect the desired boxes and move to the next ones using the “>” button.

Check open questions for which no points have been awarded#

i&e is used to check whether all the boxes in a question are empty. You can use it to check that you haven’t forgotten to correct an open question.

From “Data capture” tab

  1. Click on the “manual” button.

  2. Make sure to select “Scan”.

  3. Select the settings: i&e.

  4. Select the first ID of your open question to be checked.

  5. Make sure you start checking by being on copy 1 page 1 and click the “>” button to move to the next copy where points are missing.

  6. Correct the question and select the box to award the correct number of points.

  7. Click the “>” button until the software finds nothing more.

  8. Repeat this step for the other open questions

Calculate the results#

  1. Go back to the marking tab.

  2. Check the “update marking scale” box.

  3. Click on the “mark” button.

Associate the students list#

  1. If you modified the name of student list CSV file, you can select the other CSV list by clicking on “set file” button

  2. Select the primary key you want to sort by

  3. Click on the “Automatic” button.

Export the results#

  1. Click on the “report” tab

  2. Click on generate results

  3. To export the annotated pdfs:

  • Select “One file per student”.

  • Click on “Annotate” button.
