Start with b=1,m=0 for a basic correct/wrong scoring. Use formula=... only for advanced scoring strategies.
Scoring help

You can define scoring using either a standard scale (simple) or a formula (advanced).

1. Standard scale (simple)

The easiest way is to define scoring values for correct and wrong answers.

  • b : score for a correct answer
  • m : score for a wrong answer
  • MAX : maximum possible score
  • MIN : minimum possible score
Examples
b=1,m=0
Correct answer: +1
Wrong answer: 0
b=1,m=-1
Correct answer: +1
Wrong answer: -1
b=2,m=-0.5,MAX=2
Correct answer: +2
Wrong answer: -0.5
Score capped at 2

Values are separated with commas.

2. Formula (advanced)

For complex scoring strategies you can use a formula.

Write formulas using: 

formula=expression
Available variables
  • N : total number of answers
  • NB : number of correct answers
  • NBC : correct answers checked
  • NM : number of wrong answers
  • NMC : wrong answers checked
  • IS : 1 if simple question
  • IMULT : 1 if multiple-choice question
Operators

+ - * / ** %

Conditions
(condition ? value_if_true : value_if_false)
Formula examples
formula=NBC-NMC
formula="max(0,NBC-NMC)"
formula="(NBC==NB ? 1 : 0)"
