Auto-multiple-choice (AMC)
==========================

Auto Multiple Choice (AMC) is open-source software used to prepare, scan, mark and export exam results.

The eXamc AMC module wraps the main AMC workflow inside the web application. It is available when the AMC module is enabled for the selected exam.

The AMC page contains four tabs:

- **Project** to edit source files, update documents and run layout detection;
- **Data capture** to import and diagnose scans;
- **Marking** to compute marks and associate copies with students;
- **Results** to generate eXamc results, annotated papers and optional grading scheme reports.

The **Marking** and **Results** tabs are enabled only after scan data has been captured.
