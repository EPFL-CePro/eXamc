Exam info
============

This page is used to manage preferences of the exam and to have a global view of the stats.

.. image:: images/exam_info_global.png
   :width: 600

By clicking on the 3 dots in the right part of the screen, you can manage witch modules will be displayed on the side bar.

.. image:: images/exam_options.png
   :width: 600

By clicking on the users icon, you can manage users of the exam, by adding them as teachers, reviewers or assistants.
The search is done in the ldap database and in the app database, if your reviewer is not an EPFL member, please contact eXamc admin.

.. image:: images/manage_users.png
   :width: 600

The scale can be added by clicking on the green cross.

.. image:: images/add_scale.png
   :width: 600
