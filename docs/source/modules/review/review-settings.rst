Settings
==========================

The **Review Settings** module is used to configure online review before reviewers start correcting.

The page contains three tabs:

- **Pages groups** creates and manages the open-question groups to review;
- **Reviewers** assigns reviewers to pages groups;
- **Grading Schemes** defines predefined marking items for pages groups that use grading schemes.

An AMC project must be imported and valid before Review settings can be fully configured. If the AMC project is missing, the page displays a warning asking the user to import the project from **AMC -> Import data**.
