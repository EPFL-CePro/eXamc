Upload scans
==============

The page is dedicate to upload the non-corrected scans of a exam.
The file should be a .zip.
This step should be done after importing AMC project.

.. image:: images/review_upload.png
   :width: 800

.. warning::

   Importing scans will automatically deleting old scans. Any annotations and comments relating to them will also be deleted! If you want to keep a record of these data, you can export them (before importing them again!) via the ‘Review -> Export marked scans’ menu.
