Pages groups
------------

Review Settings - Pages Groups
--------------------------------

It is composed by a "+" button, a "save" button and a table with many entries.
The first one, "QUESTION", is dedicated to select the question.
The next entry are used to apply the numbers of pages that contain the question.

The last entries are buttons.The first one, "DELETE", is used to remove the question.
The next one is "GRADING HELP" for leaving comments on how correct the question.

After making changes, use the "SAVE" button.

   :width: 600


Grading Help
--------------

The "GRADING HELP" button is used to give the assessor comments on how to correct the question. Many tools are available
to manage the style of the comment. These comments will be displayed in the review menu during the correction process.
